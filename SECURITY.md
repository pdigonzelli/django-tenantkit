# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest  | ✅ |
| older   | ❌ |

## Reporting a Vulnerability

**Please do not open a public issue for security vulnerabilities.**

If you discover a security vulnerability, report it privately by emailing:

**pdigonzelli@gmail.com**

Include in your report:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

You can expect a response within **48 hours**. I'll keep you updated on the progress
and notify you when the issue is fixed and a release is published.

## Disclosure Policy

- I'll confirm receipt of your report within 48 hours
- I'll work on a fix and coordinate a release
- I'll credit you in the release notes (unless you prefer to remain anonymous)
- Public disclosure will happen after the fix is available
